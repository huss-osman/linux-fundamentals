# Linux Fundamentals

This repository documents my learning journey through Linux fundamentals as part of my DevOps training.

The focus is on building a strong understanding of the Linux operating system through hands-on practice, command-line usage, and problem-solving.

## Topics Covered
- Linux terminal basics  
- Core command-line utilities  
- File and directory management  
- File permissions and ownership  
- Environment variables  
- Shell basics (Bash / ZSH)  
- OverTheWire Bandit wargame (learning notes)  
- SadServers troubleshooting exercises  

## Learning Approach
- Hands-on practice in real Linux environments  
- Rewatching lessons and reinforcing concepts through small exercises  
- Documenting key takeaways, comparisons, and lessons learned  
- Learning by troubleshooting and experimentation  

## Notes
This repository focuses on **learning concepts and foundational skills**, rather than shortcuts or copy-paste solutions.
All notes are written to reflect understanding, reasoning, and personal learning outcomes.
