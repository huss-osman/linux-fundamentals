# Vim Basics

## Overview
**Vim** is a powerful text editor commonly used in **Linux environments**. It is especially useful when working on **remote servers**, where graphical editors are unavailable or impractical.

Mastering Vim improves efficiency when editing configuration files, scripts, and system files directly from the terminal.

---

## Key Concepts

### `Modes`

- **Normal mode** – default mode for navigation and commands
- **Insert mode** – used to type text (`i`)
- **Command mode** – used to save, quit, or run commands (`:`)

---

## Common Commands

> [!NOTE]
> ### Vim Quick Reference
> 
> | Command | Description |
> |----------|-------------|
> | `vim file.txt` | Open a file in Vim |
> | `i` | Enter insert mode |
> | `Esc` | Return to normal mode |
> | `:w` | Save file |
> | `:q` | Quit |
> | `:wq` | Save and quit |
> | `:q!` | Quit without saving |
> | `/word` | Search for a word in the file |
