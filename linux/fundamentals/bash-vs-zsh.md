# Bash vs Zsh

## Overview
Bash and Zsh are popular Unix shells used to interact with the operating system.

---

## Bash
- Default shell on most Linux systems
- Widely supported and script-friendly
- Common for servers and automation

---

## Zsh
- More advanced auto-completion
- Better theming and plugins
- Popular for local development environments

---

## Key Note
Shell scripts are usually written for **bash** to ensure compatibility across systems.

---

## Reflection
Understanding the differences helped me focus on Bash for scripting while appreciating Zsh enhancements for daily use.
