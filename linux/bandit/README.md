## Bandit Levels 0–12 Overview

Each level builds on the previous one, reinforcing Linux fundamentals through hands-on problem solving rather than memorisation.

---

### Level 1 → Level 2  
**File:** [level01.md](./level01.md)  
**Commands:** `ls`, `cat`  
**Focus:** Reading files in the home directory and understanding basic file access.

---

### Level 2 → Level 3  
**File:** [level02.md](./level02.md)  
**Commands:** `cat`, quoting filenames  
**Focus:** Handling filenames that begin with special characters (such as `-`).

---

### Level 3 → Level 4  
**File:** [level03.md](./level03.md)  
**Commands:** `ls -a`, `cd`, `cat`  
**Focus:** Discovering and reading hidden files in Linux directories.

---

### Level 4 → Level 5  
**File:** [level04.md](./level04.md)  
**Commands:** `file`, `cat`  
**Focus:** Identifying human-readable files among binary files.

---

### Level 5 → Level 6  
**File:** [level05.md](./level05.md)  
**Commands:** `find`, `file`  
**Focus:** Searching directories based on file size, permissions, and readability.

---

### Level 6 → Level 7  
**File:** [level06.md](./level06.md)  
**Commands:** `find` (user and group filters)  
**Focus:** Locating files using ownership and group constraints across the filesystem.

---

### Level 7 → Level 8  
**File:** [level07.md](./level07.md)  
**Commands:** `grep`  
**Focus:** Searching file contents for specific keywords efficiently.

---

### Level 8 → Level 9  
**File:** [level08.md](./level08.md)  
**Commands:** `sort`, `uniq`  
**Focus:** Identifying unique lines within large datasets using pipelines.

---

### Level 9 → Level 10  
**File:** [level09.md](./level09.md)  
**Commands:** `strings`, `grep`  
**Focus:** Extracting readable text from binary files.

---

### Level 10 → Level 11  
**File:** [level10.md](./level10.md)  
**Commands:** `base64`  
**Focus:** Recognising and decoding Base64-encoded data.

---

### Level 11 → Level 12  
**File:** [level11.md](./level11.md)  
**Commands:** `tr`  
**Focus:** Understanding character substitution and ROT13 encoding.

---

### Level 12 → Level 13  
**File:** [level12.md](./level12.md)  
**Commands:** `xxd`, `file`, `gzip`, `bzip2`, `tar`  
**Focus:** Identifying file types and systematically extracting multiple layers of compression.

---

## Skills Practised
- Navigating the Linux file system
- Reading files with unusual names
- Understanding file permissions
- Using common Linux commands
- Reading documentation to discover solutions

---

## Key Lessons
- Many security issues stem from misconfigurations
- Linux systems expose useful information if you know where to look
- Logical thinking and patience are more important than speed

---

## Reflection
Bandit reinforced Linux fundamentals in a practical way and helped build confidence when approaching unfamiliar problems using the terminal.
