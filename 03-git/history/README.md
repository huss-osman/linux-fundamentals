# Git History

![VCS](https://img.shields.io/badge/VCS-Git-F05032?logo=git&logoColor=white)
![Focus](https://img.shields.io/badge/Focus-Reading%20History-purple?logo=git&logoColor=white)

This section focuses on understanding and interpreting Git history.

Rather than just committing changes, the goal here is to read, inspect, and analyse what Git is actually storing.

Understanding history is essential for debugging, reviewing code, and collaborating safely.

## Learning Focus

- Read commit history confidently
- Understand how diffs represent changes
- Use reflog for recovery
- Interpret project evolution through commit logs
