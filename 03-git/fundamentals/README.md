This folder contains my core **Git fundamentals** notes as part of my DevOps training.

The emphasis is on understanding **how Git works internally** — snapshots, objects, branches, and history — rather than just memorising commands.

These notes focus on building a mental model of Git that transfers to real production workflows.

## Learning Focus

- Understand Git as a snapshot-based system
- Build strong mental models of commits and history
- Learn what happens behind the scenes when committing
- Avoid treating Git as a simple file tracker
- Prepare for branching, merging, and advanced workflows

These fundamentals form the base for safe collaboration, CI/CD pipelines, and infrastructure automation.
