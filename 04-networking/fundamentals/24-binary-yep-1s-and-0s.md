# Binary: Yep 1s and 0s

## Overview

This section explains the fundamentals of **binary**, the base-2 number system used by computers. It focuses on how data is represented using only **0s and 1s** and how this translates into real-world concepts like **IP addressing**.

Understanding binary is essential for working with **networking, subnetting**, and low-level system behaviour.

---

## What Is Binary?

**Binary**

A base-2 number system that uses only **two digits: 0 and 1**

### Role in Networking

- Represents all data in computers
- Forms the foundation of networking and IP addressing
- Used to store, process, and transmit information

Each digit in binary is called a **bit**.

---

## Understanding Binary Numbers

Binary numbers are based on powers of 2.

### Example:

`101010`

Each position represents a power of 2:

- 2⁰ = 1
- 2¹ = 2
- 2² = 4
- 2³ = 8
- 2⁴ = 16
- 2⁵ = 32

**Binary** helps computers convert between human-readable values and machine-level data.

---

## Binary and IP Addresses

IP addresses are stored in binary format.

### Example:

- Decimal: `192.168.1.1`
- Binary: `11000000.10101000.00000001.00000001`

Each section (octet) is **8 bits**, forming a 32-bit IPv4 address.

---

## Commands 

- `ip addr` → View IP addresses on a system
- `ifconfig` → Display network interfaces
- `ping` → Test connectivity between hosts
- `bc` → Perform binary calculations

---

## Key Takeaways

- **Binary** uses only **0s and 1s**
- Each digit is a **bit**
- Binary is the foundation of **all computing systems**
- IP addresses are represented in **binary format**
- Understanding binary is essential for **subnetting and networking**
