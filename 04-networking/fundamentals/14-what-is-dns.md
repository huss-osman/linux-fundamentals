# What Is DNS

## Overview

This section explains what **DNS (Domain Name System)** is and its role in **networking**. It focuses on how DNS translates human-readable domain names into **IP addresses** so devices can communicate.

Understanding DNS is essential for navigating the **internet**, accessing services, and troubleshooting real-world **network issues**.

---

## What Is DNS?

**DNS (Domain Name System)**  
A system that translates domain names into **IP addresses**

### Role in Networking
- Simplifies how users access websites  
- Removes the need to remember IP addresses  
- Enables communication between devices on the internet  

DNS acts as a bridge between humans and machines.

---

## Commands

These tools help interact with DNS:

- `nslookup` → Query **DNS records**  
- `dig` → Perform detailed DNS lookups  
- `ping` → Resolve domain names to **IP addresses**  
- `cat /etc/hosts` → Check local DNS overrides  

---

## Key Takeaways

- **DNS** translates domain names into **IP addresses**  
- It is essential for accessing websites and services  
- It simplifies navigation on the **internet**  
- It plays a key role in **network communication**  
- Understanding DNS helps with **troubleshooting issues**
