# Introduction To Networking

## Overview

This module introduces the core concepts of **computer networking** and how systems communicate across **networks**. It focuses on understanding how data moves between machines, how the **internet** works, and how services are exposed and accessed.

This is useful in real-world environments where understanding **DNS**, **routing**, and **network behaviour** is essential. Instead of relying on tools blindly, building a strong foundation in networking helps make sense of how systems interact and where issues occur.

## Commands

While this section is concept-focused, you will later apply your knowledge using tools such as:

- `ping` → Test **network connectivity**  
- `traceroute` → Trace the **path of network packets**  
- `nslookup` → Query **DNS records**  
- `curl` → Send **HTTP requests** to servers  

---

## Key Takeaways

- **Computer networking** underpins all modern systems and **cloud infrastructure**  
- Every request follows a path: **client → DNS → network → server → response**  
- Understanding networking is essential before working with **AWS, Docker, and Kubernetes**  
- Strong fundamentals make **debugging and troubleshooting** significantly easier
