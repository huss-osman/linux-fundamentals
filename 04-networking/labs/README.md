# Networking Labs

![Networking](https://img.shields.io/badge/Networking-Computer%20Networks-607D8B?logo=cloudflare&logoColor=white)
![Tools](https://img.shields.io/badge/Tools-ping%20%7C%20traceroute%20%7C%20nslookup-blue)
![Practice](https://img.shields.io/badge/Practice-Hands--on-orange)

This folder contains **hands-on networking labs** focused on practical troubleshooting and network investigation skills.

The focus is on applying networking fundamentals through real command-line investigation and troubleshooting workflows.

Labs emphasise understanding how systems communicate, how traffic flows across networks, and how to debug connectivity issues in real-world environments.

---

## Topics Covered

- [Troubleshoot Like a Pro](./27-troubleshoot-like-a-pro.md)  
  Developing a structured approach to diagnosing and resolving networking issues.

- [Troubleshooting with ping, traceroute & nslookup](./28-troubleshoot-with-ping-traceroute-nslookup.md)  
  Using core networking tools to investigate connectivity, latency, and DNS resolution.

---

## Learning Focus

- Develop a structured troubleshooting mindset  
- Use real networking tools to diagnose issues  
- Understand how traffic flows across networks  
- Identify where failures occur (DNS, routing, or host)  
- Build confidence debugging real-world connectivity problems
