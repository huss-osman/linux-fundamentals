# Networking Assignments

![Networking](https://img.shields.io/badge/Networking-Computer%20Networks-607D8B?logo=cloudflare&logoColor=white)
![Focus](https://img.shields.io/badge/Focus-Networking%20Fundamentals-purple?logo=cloudflare&logoColor=white)
![Assignments](https://img.shields.io/badge/Assignments-1-darkgreen)

This assignment focuses on reinforcing **core networking fundamentals** through structured, real-world challenges.

Assignment must:

- Demonstrate understanding of networking concepts (DNS, IP, routing, ports)  
- Show clear configuration steps and reasoning  
- Validate connectivity using real tools  
- Reflect real-world infrastructure setup  

> [!TIP]  
> Attempt the assignment independently before searching for solutions.  
> Use tools like `ping`, `curl`, `nslookup`, and browser testing to verify behaviour.

---

### Assignment — Domain + EC2 + DNS (NGINX)  
**File:** [30-domain-ec2-dns.md](./30-domain-ec2-dns.md)  
**Concepts:** DNS, IP addressing, ports, routing, cloud networking  

**Focus:** Connecting infrastructure to the internet and validating end-to-end communication.

**Requirements**

- Buy and configure a domain (Cloudflare or Route53)  
- Launch an EC2 instance and install NGINX  
- Configure DNS (A record → EC2 public IP)  
- Validate access via browser and networking tools  

---

## Skills Reinforced

- DNS configuration and propagation  
- Public IP addressing and routing  
- Port-based communication (HTTP)  
- Cloud networking fundamentals (EC2 + security groups)  
- End-to-end connectivity testing  

---

## Learning Outcome

These assignments bridge the gap between networking theory and real infrastructure.

By applying DNS, IP addressing, and routing in a practical setup, they reinforce how systems are exposed and accessed over the internet in real-world environments.
