## Overview
Exit codes indicate success or failure of commands and scripts.

---

## Key Concepts

- `0` means success
  
- Non-zero means failure
  
- Used heavily in automation

---

## Example

```bash
exit 1
```

## Key Takeaways

- Exit codes control flow

- Essential for CI/CD

- Silent failures are dangerous

## Reflection
Exit codes explained how tools communicate in pipelines.
