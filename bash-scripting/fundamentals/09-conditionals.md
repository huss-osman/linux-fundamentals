# Conditionals

## Overview
Conditionals allow scripts to make decisions based on conditions.

---

## Key Concepts

- Based on exit codes
  
- Commands return success or failure
  
- Tests evaluate conditions

---

## Example

```bash
if [ -f file.txt ]; then
  echo "File exists"
fi
```

---

## Key Takeaways

- Conditionals rely on exit status

- Spacing matters

- Defensive checks improve safety

---

## Reflection
Conditionals made scripts more intelligent and robust.
