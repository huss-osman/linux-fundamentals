# Functions

## Overview
Functions group reusable logic inside scripts.

---

## Key Concepts

- Improve structure
  
- Reduce duplication
  
- Support local scope

---

## Example

```bash
greet() {
  echo "Hello $1"
}
```

---

## Key Takeaways

- Functions improve readability

- Essential for larger scripts

- Encourage modular design

---

## Reflection
Functions helped scripts scale without becoming messy.
