# Shell Options (set)

## Overview
Shell options improve script reliability and safety.

---

## Key Concepts

- Fail fast
  
- Catch undefined variables
  
- Debug execution

---

## Options

```bash
set -e
set -u
set -x
set -eux
```

## Key Takeaways

- set -eux is production-safe

- Prevents silent bugs

- Improves debugging

## Reflection
Strict mode made scripts predictable and trustworthy.
