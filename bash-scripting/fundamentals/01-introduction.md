# Introduction to Bash

## Overview
Bash (Bourne Again Shell) is a command-line shell and scripting language used to interact with Linux systems and automate tasks.

In DevOps environments, Bash is used extensively for automation, system configuration, CI/CD pipelines, and troubleshooting.

---

## Key Concepts

- Bash executes commands exactly as they would run in an interactive terminal
  
- Scripts follow the same execution rules as shell commands
  
- Bash prioritises simplicity and composability

---

## Examples

```bash
echo "Hello from Bash"
```

---

## Key Takeaways

- Bash is foundational to Linux automation

- Scripts reflect real system behaviour

- Understanding Bash improves system-level reasoning
