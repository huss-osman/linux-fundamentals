# Loops

## Overview
Loops allow repeated execution of commands.

---

## Key Concepts

- `for` iterates over items
  
- `while` runs while a condition is true

---

## Example

```bash
for file in *.txt; do
  echo "$file"
done
```

---

## Key Takeaways

- Loops reduce duplication

- Useful for batch operations

- Careful construction is required

---
