This folder contains **hands-on Docker labs** completed as part of my DevOps training.

The focus is on troubleshooting containers, debugging service communication, and understanding how Docker behaves in real environments.

Labs emphasise investigation, logs, networking, and container debugging workflows.

---

## Topics Covered

- [Debugging MySQL Errors](./20-debugging-mysql-errors.md)  
  Investigating and fixing container issues related to MySQL configuration and startup failures.

- [Debugging Networking Issues](./25-debugging-networking-issues.md)  
  Troubleshooting Docker networking and communication problems between services.

---

## Learning Focus

- Develop structured Docker troubleshooting habits  
- Learn how to investigate container failures  
- Understand networking and service communication issues  
- Build confidence debugging multi-container environments  
- Strengthen operational and troubleshooting skills
